# デバイスリサーチ シミュレーター V2 (Device Research Simulator)

## 概要 (Overview)
厚労科研検索用デバイスリサーチプロジェクトのシミュレーターです。

## 使用方法 (How to Use)
1. `simulator_v2.html` をブラウザで開いてください。
2. デバイスを選択すると、対応する機器イメージと価格、機能比較が表示されます。
3. 下部の事例紹介（Case Study）セクションでは、各自治体・医療機関の取り組みを確認できます。

## 構成 (Structure)
- `simulator_v2.html`: メインのHTMLファイル
- `images/`: デバイス画像フォルダ (Yamasa, Tanita, Omron, Fitbit)
