# スライド2: システム構成・運用フロー

**1. スマホユーザー**
*   **デバイス**: Omron または Fitbit
*   **運用**: アプリで自動連携
*   **メリット**: データは日々自動送信されるため手間なし

**2. 非スマホユーザー**
*   **デバイス**: **Omron 必須** (※Fitbitは履歴確認困難なため不可)
*   **日常**: **「みんなの手帳」** に歩数を転記 (週1回以上)
*   **来院時**: 手帳を見ながら施設タブレットに入力

# スライド3.5: 製品仕様と選定の妥当性

| 項目 | Omron HJA-405T | Fitbit Inspire 3 |
| :--- | :--- | :--- |
| **画像** | (製品画像) | (製品画像) |
| **サイズ・重量** | 33×78×12mm / **約28g** | 39×19×12mm / **約18g** |
| **電源** | ボタン電池 **(CR2032)** (約3ヶ月) | 充電式 (最大10日間) |
| **メモリ** | 画面7日 / 内部14日 | 詳細7日 / 30日保存 |
| **選定の妥当性** | **「非スマホ層の命綱」**<br>本体画面で過去データを振り返れる数少ない機種。<br>手帳転記運用には必須。 | **「装着ストレスフリー」**<br>圧倒的に軽く、睡眠中も気にならない。<br>30日データ保持で同期忘れリスクを極小化。 |

# スライド5: 推奨事項・まとめ

*   **予算感 (定価ベース試算)**
    *   200名で約306万円、500名で約705万円。

    *   *規模: 全国 **3,000施設以上** 導入 / 分散型臨床試験(DCT)支援実績多数*
